## 🚀 快速启动

### 编译和运行
```bash
# 1. 安装Qt6 (如果尚未安装)
brew install qt

# 2. 编译应用
qmake && make

# 3. 运行应用
./qt_pomodoro.app/Contents/MacOS/qt_pomodoro
```

### 主要功能特性
- ✅ **智能计时**: 25分钟工作 + 5分钟休息
- ✅ **音量调节**: 0-100%可调节提示音
- ✅ **主题系统**: 深色/浅色主题切换
- ✅ **系统托盘**: 精简显示（只显示分钟）
- ✅ **浮动窗口**: 独立计时显示
- ✅ **自动锁屏**: 休息时间自动锁屏
- ✅ **主题记录**: 工作主题记录和导出
